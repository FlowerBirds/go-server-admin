# go-server-admin
服务器监控软件，目前适用于CentOS7,主要用于服务器上命令执行监控记录，记录操作日志，跟踪执行命令

## 功能列表
1. 记录服务器上操作的命令日志，包括执行用户、客户端IP、执行时间、执行内容；
2. portal页面展示执行记录，可根据服务器IP进行筛选
3. ssh首次登录后执行命令，通过FeiQ通知
